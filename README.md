# Weather App 🌤️

A simple Weather App that shows the current weather of any city using a weather API.  
Users can enter a city name and get live weather information instantly.

## 🚀 Features
- Search weather by city name
- Shows temperature and weather condition
- Clean and simple user interface
- Responsive design

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

## 📂 Project Structure
weather-app
│
├── index.html
├── style.css
├── script.js
└── README.md

## ⚙️ How It Works
1. User enters a city name.
2. JavaScript sends a request to the Weather API.
3. The API returns data in JSON format.
4. The weather information is displayed on the webpage.
